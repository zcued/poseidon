# Badge

```js
import Badge from '@zcool/badge'
```

```.jsx
() => (
  <div style={{ background: '#f0f4f5' }}>
    <Badge count={2} dot={true}><span>徽标</span></Badge>
    <br />
    <Badge count={2}><span>徽标</span></Badge>
    <br />
    <Badge count={23} showDot={true}><span>徽标</span></Badge>
    <br />
    <Badge count={2} dot={true} offset={[10, 10]}><span>徽标</span></Badge>
  </div>
)
```

## props

| props           | type             | default |
| :-------------: | :---------------:| :-----: |
| count           | ReactNode        | true    |
| dot             | boolean          | false   |
| offset          | [number, number] |         |
